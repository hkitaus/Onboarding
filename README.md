## 社群守則

1. **互相尊重** - 尊重每位成員，不論背景、經驗或觀點如何。
2. **專業交流** - 保持討論的專業性，避免不必要的爭論。
3. **實用分享** - 分享有用的資訊和經驗，避免垃圾內容或廣告。
4. **保護私隱** - 尊重他人私隱，不要未經許可分享個人資料。
5. **積極參與** - 鼓勵積極參與討論，但請避免發送過多無關訊息。
